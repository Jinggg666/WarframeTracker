# Gameplay Tutorials · 其他游戏玩法教程

Guides for missions, modes, and mechanics that aren't tied to a specific weapon or Warframe.

**Suggested topics:**
- Steel Path / Endurance
- Eidolon hunts
- Profit-Taker / Orb Mother
- Railjack basics
- Open-world bounties (Plains, Vallis, Cambion Drift, Zariman, Duviri)

**Adding a tutorial:**
- Drop a Markdown file (`my-guide.md`) in this folder.
- Wire it into `public/app.js` (add an entry to the section's tutorial list — that part is currently a placeholder).
